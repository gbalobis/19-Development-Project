How to access our game:
Through github:
1.	copy and paste this link on your internet browser:
https://github.com/gbalobis/19-Development-Project.git
2.	download all the files through github
3.	run programs through preferred method of coding (terminal/IDE)
How to play out game:
1.	Use WASD keys to navigate up, left, down, right respectively
2.	Press enter key after every input
3.	When two tiles of the same value collide a tile which is the sum of the two tiles that collided will form.
4.	Keep on using WASD until either you make the number “2048” or there is no longer a possible way for two tiles to collide.
